Adding CSS Style to HTML Structure

What is CSS?

CSS is the decoration of the walls, displaying the HTML in the way desired. It is applied as the law of the website with a concept called inheritance . What this means is that properties are applied to a specificed element as well as any continuing ones, essentially the entire page. CSS is a great way to avoid repetition so programmers don't have to write the same code over and over. It also avoids problems that can arise from repeated code since every code needs to be adjusted. Using CSS will just change everything in one go.

How does CSS control HTML?

With a selection, declaration, value and property. The section could be a heading tag or div class tag. This concept has been explained in this video.

There are three methods of adding CSS Styling to a Webpage:
  • Method 1: Writing CSS in the head tag of your HTML
  • Method 2: Link your HTML to separate CSS file
  • Method 3: Write your style inline with your HTML
